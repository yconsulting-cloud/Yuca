2. Données collectées

Nous collectons les données suivantes :

2.1 Données fournies volontairement

Lorsque vous utilisez notre formulaire de contact ou notre chatbot :

  • Nom
  • Adresse email
  • Numéro de téléphone (optionnel)
  • Activité professionnelle
  • Description de votre projet

2.2 Données collectées automatiquement

Via Google Analytics, nous collectons de manière anonymisée :

  • Pages visitées
  • Durée de visite
  • Type d'appareil et navigateur
  • Pays d'origine (non précis)

3. Finalités du traitement

4. Destinataires des données

Vos données peuvent être partagées avec :

  • Brevo (ex-Sendinblue) : pour la gestion des contacts et l'envoi d'emails (hébergement UE)
  • Google Analytics : pour la mesure d'audience (données anonymisées)
  • Vercel : hébergeur du site (États-Unis, clauses contractuelles types)
  • Anthropic : pour le chatbot IA (États-Unis, clauses contractuelles types)

Nous ne vendons jamais vos données personnelles à des tiers.

5. Transferts hors UE

Certaines données peuvent être transférées vers des pays hors de l'Union Européenne (États-Unis notamment). Ces transferts sont encadrés par :

  • Le Data Privacy Framework (DPF) pour les entreprises certifiées
  • Des clauses contractuelles types approuvées par la Commission Européenne

6. Vos droits

Conformément au RGPD, vous disposez des droits suivants :

  • Droit d'accès : obtenir une copie de vos données
  • Droit de rectification : corriger vos données inexactes
  • Droit à l'effacement : demander la suppression de vos données
  • Droit à la limitation : limiter le traitement de vos données
  • Droit à la portabilité : recevoir vos données dans un format structuré
  • Droit d'opposition : vous opposer au traitement de vos données

Pour exercer ces droits, contactez-nous à : yuca.consulting@gmail.com

Vous pouvez également introduire une réclamation auprès de la CNIL : www.cnil.fr

7. Cookies

7.1 Qu'est-ce qu'un cookie ?

Un cookie est un petit fichier texte déposé sur votre appareil lors de la visite d'un site web.

7.2 Cookies utilisés sur ce site

7.3 Gestion des cookies

Vous pouvez à tout moment désactiver les cookies via les paramètres de votre navigateur :

  • Google Chrome
  • Mozilla Firefox
  • Safari
  • Microsoft Edge

8. Sécurité

Nous mettons en œuvre les mesures techniques et organisationnelles appropriées pour protéger vos données personnelles :

  • Connexion HTTPS chiffrée
  • Accès restreint aux données
  • Hébergement sécurisé
  • Clés API protégées côté serveur

9. Chatbot IA

Notre site utilise un chatbot alimenté par l'intelligence artificielle (Claude, par Anthropic). Lorsque vous interagissez avec le chatbot :

  • Vos messages sont envoyés à l'API d'Anthropic pour générer une réponse
  • Les conversations ne sont pas conservées par Yuca après votre session
  • Si vous soumettez le formulaire de contact via le chatbot, vos données sont enregistrées dans notre CRM (Brevo)
